PrismRAG retrieval plug-in

Fourth swappable port — vector search + taxonomy remap in ~20 lines, not a benchmark fork.

pip install "chorusgraph[retrieval]"

from chorusgraph.compose import ChorusStack, PrismRAGRetrievalBackend

backend = PrismRAGRetrievalBackend(embedder=..., mapping=YOUR_MAPPING)
backend.index(YOUR_CORPUS)  # opt-in: partition="kb_markdown", version="..."

stack = ChorusStack.defaults(tenant_id="acme").with_retrieval(backend)
# stack.warm_retrieval(partition="kb_markdown")  # 1.1.0 — process boot
retrieve_node = stack.to_retrieve_handler(
    topic="policy", top_k=6,
    # rerank_policy="vectors_only",  # 1.1.0 optional speed path
)

Vector search works without a license; taxonomy remap requires PrismRAG Pro license. 1.1.0 adds optional warm chunk vectors for RAG latency. Full guide: INSTALL.md
